# loan-approval-prediction
A Machine Learning project that predicts loan approval status using Logistic Regression. Includes data preprocessing, label encoding, feature scaling, model training, evaluation, user input prediction, and model saving with Joblib

## Dataset
- Loan Approval Dataset
- Target Variable: Loan_Status

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Algorithm
- Logistic Regression

## Features
- Data Cleaning
- Label Encoding
- Feature Scaling
- Train-Test Split
- Model Training
- User Input Prediction
- Model Saving using Joblib

## Results
- Accuracy: 92%
